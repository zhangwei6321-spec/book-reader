## 🛠 技术栈

- 纯 HTML / CSS / JavaScript，无框架依赖
- localStorage 本地数据存储
- jsPDF 客户端 PDF 生成
- Notification API 桌面提醒
- GitHub Pages 静态部署

## 📂 项目结构

```
book-reader/
├── index.html              ← 总主页
├── apps/
│   ├── habit-tracker/      ← 习惯打卡
│   └── timer-app/          ← 专注计时器
├── 题库/                   ← 公基刷题库
├── txt仿真阅读/             ← 仿真书架阅读器
├── video-parser/           ← 短视频去水印

├── 证件照/                  ← 证件照处理
├── castle-escape/          ← 妖怪迷城
└── games/                  ← 棋类游戏中心
```
