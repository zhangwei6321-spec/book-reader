## 🛠 技术栈

- 纯 HTML / CSS / JavaScript，零框架依赖
- Canvas 渲染游戏棋盘 + 图像处理
- AI 抠图：浏览器端 WebAssembly（`@imgly/background-removal`）
- localStorage 存储进度与设置
- GitHub Pages 静态托管
